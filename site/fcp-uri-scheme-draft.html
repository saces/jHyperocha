FCP URI scheme - draft

What is this?

This document describes the fcp uri scheme.
This document is an early working draft and is subject to change.

FCP uri scheme

fcp://<user>:<password>@<host>:<port>/<freenetkey>;<connectionparams>?<commandparams>
The fcp uri scheme looks like the http uri scheme. Username and password are defined for future use, but currently ignored. If the host or port is not given, the implementation should default to 127.0.0.1:9481
Parameter names should be case insensitive.
  • connection parameters. have only effect if new connections are created
    FCPLog
    if the parameter is set, any value turns logging on)
    Timeout
    set timeout to the amount of seconds
  • command parameters
    Priority
    the priority class for the request (0...6)
    MaxRetries
    set the max retry count. -1 retries forever
    Type
    overwrite the mime type
    MaxSize
    the max size of the resulting data
